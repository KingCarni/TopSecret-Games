This folder is optional. It exists so Codex has a safe place to wire curated public images later.

Recommended future files:
- nova-preview.png

Rules:

- No internal dashboards with secrets
- No private repo screenshots
- Crop tightly and present clearly

If you add real images later, set project.media.imageSrc in:
website/lib/topsecret/site-data.ts

Example:
